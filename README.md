#  Bank Management System

A simple **Bank Management System** built with **Python** using **Object-Oriented Programming (OOP)**. This project allows users to perform basic banking operations through a menu-driven interface while demonstrating Python fundamentals such as classes, methods, loops, file handling, and input validation.

---

##  Features

- ✅ Create a bank account
- ✅ Deposit money
- ✅ Withdraw money
- ✅ Check account balance
- ✅ View account details
- ✅ View transaction history
- ✅ Save account data to a file
- ✅ Input validation
- ✅ Menu-driven interface

---

## ▶ How to Run

1. Clone the repository:

```bash
git clone https://github.com/ismailkhan-dev/Bank-Management-System.git
```

2. Open the project folder.

3. Run the program:

```bash
python bank_management_system.py
```

4. Follow the on-screen menu.
